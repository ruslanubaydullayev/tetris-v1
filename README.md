## Classic 2D Games

Simple, user-friendly website for playing classic 2D games directly in the browser.

### Features

- **Only 2D games**: No 3D, no heavy graphics.
- **Clean layout**: Game list on the side, focused play area.
- **Snake included**: Fully playable Snake implementation using HTML canvas.
- **Keyboard controls**:
  - **Arrow keys / WASD**: Move.
  - **P**: Pause / resume.
  - **R**: Restart.

### Getting started

- **Open directly**: Double-click `index.html` in your file manager to play locally.
- **Or run a tiny dev server**:

```bash
cd /Users/ruslan/Desktop/ai
npm install
npm run start
```

Then open the printed URL in your browser.

### Structure

- `index.html` – main page and layout.
- `styles.css` – styling, responsive layout, dark theme.
- `script.js` – Snake game logic and basic game switching.
- `package.json` – helper scripts for running a local static server.
